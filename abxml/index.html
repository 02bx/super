Crate abxml [−] [src]

Library that decodes the binary documents contained on an APK (both resources.arsc and binary XMLs). It exposes also structures to query this binary files on a structured way. For example, it's possible to check which chunks of data a document contains, and perform specific queries depending on the type of chunk.
